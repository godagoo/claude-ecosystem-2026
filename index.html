Claude Ecosystem Guide

Important Distinction

Chrome Extension vs --chrome Flag

Two ways to automate browsers. Different purposes.

Claude in Chrome

Browser Extension

  • Everyday browser tasks
  • Form filling, scheduling
  • Data extraction
  • Side panel interface
  • Uses logged-in sessions
  • For everyone
Non-developers
Claude Code --chrome

Terminal Flag

  • Developer testing
  • Web app debugging
  • Console log reading
  • Runs from command line
  • Built for devs
  • Works with extension
Developers

They're complementary. Developers: use both. Non-developers: Chrome extension is what you want.

File automation for non-developers. Desktop app, separate tab.

Security Warning

PromptArmor Vulnerability

Disclosed BEFORE launch. Anthropic launched anyway.

  • Hidden prompt injection in files
  • Files exfiltrated via API
  • Even Opus 4.5 falls for it

Recommendation: Don't give sensitive files. Wait for fix.
